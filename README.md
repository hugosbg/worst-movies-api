# API Rest

> Lista de indicados e vencedores da categoria Pior Filme do Golden Raspberry Awards

### Usage

```shell
$ git clone https://github.com/hugosbg/worst-movies-api.git
$ cd worst-movies-api/
$ yarn
$ cp .env.example .env
$ yarn start:dev
```

### Test

```shell
$ yarn test
```

### Documentation

Swagger: http://localhost:3000/api/doc
